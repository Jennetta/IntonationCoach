About Intonation Coach


This app was built in 4.5 weeks as a final project for Hackbright Academy's Software Engineering Fellowship.

Technology Stack

Intonation Coach has a Python backend running on a Flask server, with an SQLite database and SQLAlchemy. The frontend is written in Javascript with jQuery and AJAX, and HTML/CSS with Bootstrap.

Pitch analysis is done with Michael McAuliffe's python-praat-scripts package, which harnesses the pitch detection algorithm of the phonetic analysis program Praat. The pitch data extracted from this process is then smoothed, formatted, serialized as JSON, and passed back to the frontend to be displayed in the graph.

Graphs are created with NVD3, with additional animation using D3. Transcripts below each graph are aligned with SVG.

Audio is captured from the browser with the help of Matt Diamond's Recorder.js and played through the Web Audio API. User recordings are encoded as data URLs and passed to the server, where they are decoded using base-64 decoding and written to WAV files to be fed into the pitch analysis module. Audio and pitch data for each user recording are also saved to the database; on each tab load, the relevant recordings for that sentence are fetched from the database. Their pitch data is added to the graph while audio data is given an object URL and loaded into a buffer to be played.
